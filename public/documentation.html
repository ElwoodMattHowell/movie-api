Movie API Documentation
URL endpoints and request methods
Method URL Request Response Parameters
GET /movies Request a list of all movies Returns a JSON of all movies
GET /movies/title Data about a single movie A JSON with description and details about a single movie
GET /genres/genre name A list of all movies in a specific genre A JSON of all movies in a specific title
GET /directors/director name Biographical information of a specific director on list A JSON of biographical information of a specific director
POST /users Allow new users to register JSON of new user info with a unique ID number { name: "Billy-Bob Joe", age: "100", username:"BBJ123!", email: "email@gmail.com", password: "123456" }
PUT /users/username Update personal information JSON of updated personal information { name: "Billy-Bob Joe", age: "100", username:"BBJ123!", email: "email@gmail.com", password: "123456" }
POST /users/username/movies/MovieId Add a movie to list of favorites JSON of user information
DELETE /users/username/movies/MovieId Remove a movie from users list of favorites JSON of user information
DELETE /users Remove user "Username" succesfully deleted
