This API will allow users to choose movies included in the API and add them to a list of their favorite movies. New users will be able to register and existing users can update their user information. Movie information includes a biography of the movie's director.

Movie API Documentation
URL endpoints and request methods
Method URL Request Response Body Authentication
POST /users Allow new users to register JSON of new user info {
name: "Billy-Bob Joe",
age: "100",
username:"BBJ123!",
email: "email@gmail.com",
password: "123456"
} 		None
POST /login Login and generate bearer token JSON of new user info and a bearer token to use to authenticate further requests {
username:"BBJ123!",
password: "123456"
} 		Username and Password needed
PUT /users/username Update personal information JSON of updated personal information {
name: "Billy-Bob Joe",
age: "100",
username:"BBJ123!",
email: "email@gmail.com",
password: "123456"
} 		Bearer Token
GET /movies Request a list of all movies Returns an array of all JSONs of movie information Bearer Token
GET /users Request a list of all users Returns an array of all JSONs of user information Bearer Token
GET /movies/title Data about a single movie A JSON with description and details about a single movie Bearer Token
GET /genres/genre name A list of all movies in a specific genre A JSON of all movies in a specific title Bearer Token
GET /directors/director name Biographical information of a specific director on list A JSON of biographical information of a specific director Bearer Token
POST /users/username/movies/MovieId Add a movie to list of favorites JSON of user information Bearer Token
DELETE /users/username/movies/MovieId Remove a movie from users list of favorites JSON of user information Bearer Token
DELETE /users Remove user "Username" succesfully deleted Bearer Token
