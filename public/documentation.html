Movie API Documentation

URL endpoints and request methods
Method URL Request Response Parameters
GET /movies Request a list of all movies Returns a JSON of all movies
GET /movies/title Data about a single movie A JSON with description and details about a single movie
GET /genres/genre type A list of all movies in a specific genre A JSON of all movies in a specific title
GET /directors/ A list of all directors on list A JSON of all directors on list
GET /directors/director name Biographical information of a specific director on list A JSON of biographical information of a specific director
POST /users/newuser Allow new users to register JSON of new user info with a unique ID number
PUT /users/username/update Update personal information JSON of updated personal information { name: "Billy-Bob Joe", age: "100", username:"BBJ123!", email: "email@gmail.com", password: "123456" }
GET /users/username/favorites List of favorite movies with info JSON of favorite movies with info
POST /users/username/favorites Add a movie to list of favorites "Movie Title" succesfully added { title: "Raiders of the Lost Ark", Year: "1981" Director: Steven Spielberg, Genre: "Awesomeness" }
Delete /users/username/favorites Remove a movie from users list of favorites
DELETE /users/deregister Remove user "Username" succesfully deleted
